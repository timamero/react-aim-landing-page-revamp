# Finance App Landing Page (Revamp)

A modernized iteration of a mock finance company landing page demonstrating strategic use of AI tools for efficient development while maintaining code quality and maintainability.

## Overview

This project is a complete revamp of the original [Finance App Landing Page](https://github.com/timamero/react-aim-landing-page), modernizing the tech stack, improving design, and optimizing file structure. More importantly, it showcases a professional, human-in-the-loop approach to leveraging AI tools throughout the development lifecycle.

## Key Updates

- **Build Tool:** Migrated from Create React App (CRA) to Vite for faster development and optimized production builds
- **Animation Library:** Upgraded to anime.js for more robust, flexible animation capabilities
- **File Structure:** Reorganized for better scalability, maintainability, and separation of concerns
- **Design:** Refined visual hierarchy, improved responsive behavior, and enhanced user experience

## Development Approach: AI-Assisted, Human-Verified

This project demonstrates how to effectively use AI tools as a force multiplier in development while maintaining professional standards. The approach follows an iterative, human-in-the-loop workflow:

### Phase 1: Planning & Guidelines

- **Project Specifications:** Defined clear requirements, design goals, and technical constraints
- **AI Guidelines:** Established guardrails for AI code generation, including coding standards, performance expectations, and security requirements
- **Architecture Decisions:** Used AI to explore design patterns and evaluate trade-offs before implementation

### Phase 2: Iterative Development Cycle

Each feature follows this workflow:

1. **Generation:** AI generates code based on detailed specifications and guidelines
2. **Verification:** Manual review of generated code for correctness, security, and alignment with standards
3. **Refinement:** Manual refactoring or AI-assisted improvements based on findings
4. **Testing:** Validate functionality and performance
5. **Integration:** Merge into the main codebase

## How AI Enhanced This Project

### Code Generation & Refactoring

- Accelerated boilerplate and component scaffolding while maintaining custom logic integrity
- Batch-refactored code for improved readability and consistency across the project
- Generated utility functions and helper modules with human oversight

### Debugging & Root Cause Analysis

- Rapid hypothesis testing for bugs using AI-assisted analysis
- Systematic error investigation reducing time to resolution
- Identification of edge cases and potential improvements

### Architecture & Design Decisions

- Explored multiple file structure approaches before implementation
- Evaluated animation library options and integration strategies
- Designed responsive layouts with AI-assisted media query generation (verified manually)

### Development Speed

- Reduced time spent on repetitive tasks without sacrificing code quality
- Faster iteration cycles while maintaining accuracy through verification steps
- Increased productivity without compromising security or maintainability

## Key Principles

✓ **Accuracy First:** All AI-generated code is reviewed and tested before integration  
✓ **Maintainability:** Code follows consistent standards and is documented appropriately  
✓ **Security:** Dependency updates are verified, and no security shortcuts are taken  
✓ **Human Judgment:** Complex decisions remain under human control; AI assists, doesn't replace  
✓ **Transparency:** Process is documented to show intentional AI usage

## Tech Stack

- **Styling:** CSS
